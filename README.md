This project performs Exploratory Data Analysis (EDA) on a retail dataset to extract meaningful business insights related to customers, products, and sales trends.

Tech Stack

Python
Pandas
NumPy
Matplotlib
Seaborn

Data Cleaning

Removed missing Customer IDs
Handled missing product descriptions
Filtered invalid transactions (negative values)
Converted date column to proper datetime format

Feature Engineering

Created TotalPrice = Quantity × UnitPrice
Extracted Date and Month from transaction data

Analysis

Top 10 customers by spending
Top products by quantity and revenue
Daily and monthly sales trends
Country-wise order distribution
RFM (Recency, Frequency, Monetary) analysis

Visualizations

Bar charts for top customers, products, and countries
Line plots for revenue trends
Correlation heatmap
Distribution plots (log-transformed and outlier-removed)

Key Insights

Identified top-performing products and customers
Observed sales trends over time
Found peak sales periods
Analyzed customer purchasing behavior

Outputs

Cleaned dataset
RFM analysis file
Visualization images
Business insights report

Business Value

Helps identify high-value customers
Supports better product and sales strategies
Enables data-driven decision making
